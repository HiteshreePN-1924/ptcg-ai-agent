# 🧠 PTCG AI Battle Agent

## Overview
This project is an intelligent decision-making AI for the Pokémon Trading Card Game (PTCG).  
It evaluates game states and selects optimal moves using a scoring-based strategy engine with controlled randomness for realism.

---

## 🧩 How it Works

The AI follows a structured pipeline:

1. Reads game state (HP, energy, opponent status)
2. Generates possible moves
3. Scores each move using weighted strategy rules
4. Adds controlled randomness for human-like behavior
5. Selects the best action

---

## ⚙️ Key Features

- Dynamic move evaluation system
- Strategy-based decision engine
- Adaptive scoring logic
- Controlled randomness for realism
- Lightweight and fast simulation

---

## 🧠 AI Strategy Logic

The agent evaluates:

- Attack potential based on energy and opponent HP
- Survival probability based on current HP
- Heal value when low HP
- Strategic retreat decisions

## 🛠 Tech Stack

- Python
- Rule-based AI system
- Strategy scoring engine
